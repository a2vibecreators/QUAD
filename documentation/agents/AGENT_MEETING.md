## Overview

The **Meeting Agent** manages team meetings end-to-end: scheduling, minute-taking, action item tracking, and follow-up automation. It transforms meetings from time-consuming interruptions into valuable decision-making forums with automatic documentation and accountability.

### Core Capabilities

1. **Intelligent Meeting Scheduling** - Find availability, book rooms, send invites
2. **Automated Meeting Notes** - Record, transcribe, summarize key decisions
3. **Action Item Tracking** - Extract todos, assign owners, follow up
4. **Calendar Integration** - Sync with Google Calendar, Outlook, Slack
5. **Smart Meeting Analytics** - Attendance, decision quality, follow-up rates

```yaml

  enabled: true

  enabled: true
  provider: "openai-whisper"  # or "rev", "otter"
  auto_generate_summary: true
  extract_action_items: true
  extract_decisions: true

action_items:
  enabled: true
  jira_integration: true
  jira_project_key: "DEV"
  jira_issue_type: "Task"
  linear_integration: false
  github_integration: false

notifications:
  slack_enabled: true
  email_enabled: true
  slack_channel: "#engineering"
  send_transcript: true
  send_action_items: true

recording:
  enabled: true
  auto_join: true  # Bot auto-joins meetings
  storage_provider: "google-drive"  # or "s3", "azure"
  retention_days: 365

ignore:
  - "1-on-1 meetings"  # Don't record confidential 1-on-1s
  - "#human-resources"  # Don't record HR channel meetings
  - "Board meetings"  # Executive meetings (manual recording)
```
